## 第四章 栈
栈是一种遵从后进先出（LIFO）原则的有序集合。

## 第五章 队列和双端队列
队列是遵循先进先出（FIFO）原则的一组有序的项。队列在尾部添加新元素，并从顶部移除元素。  
双端队列是一种允许我们同时从前端和后端添加和移除元素的特殊队列。  

## 第六章 链表
链表存储有序的元素集合，但不同于数组，链表中的元素在内存中并不是连续放置的。每个元素由一个存储元素本身的节点和一个指向下一个元素的引用（指针/链接）组成。  

## 第七章 集合
集合是由一组无序且唯一的项组成的。
